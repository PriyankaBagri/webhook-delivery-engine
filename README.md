# 🚀 Full-Stack Webhook Delivery Engine

A real-time webhook tracking and monitoring dashboard built with Node.js, Express, SQLite3, and HTML/CSS. 

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** SQLite3
- **Frontend:** HTML5, Tailwind CSS / Custom Dark Theme

## 🚧 Status: Work In Progress (WIP)
Currently setting up the connection between the SQLite local database logs and the frontend dark-mode dashboard interface. Bypassing CORS policies and mapping local server routes.
